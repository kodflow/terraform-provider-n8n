# Trigger Nodes Examples

This directory contains examples for **Trigger** category nodes.

Total nodes in this category: **25**

## Nodes in this Category

- **Acuity Scheduling Trigger** (`acuitySchedulingTrigger`) - Handle Acuity Scheduling events via webhooks
- **Bitbucket Trigger** (`bitbucketTrigger`) - Handle Bitbucket events via webhooks
- **Cal.com Trigger** (`calTrigger`) - Handle Cal.com events via webhooks
- **Calendly Trigger** (`calendlyTrigger`) - Starts the workflow when Calendly events occur
- **Email Trigger (IMAP)** (`emailReadImap`) - Triggers the workflow when a new email is received
- **Error Trigger** (`errorTrigger`) - Triggers the workflow when another workflow has an error
- **Eventbrite Trigger** (`eventbriteTrigger`) - Handle Eventbrite events via webhooks
- **Facebook Lead Ads Trigger** (`facebookLeadAdsTrigger`) - Handle Facebook Lead Ads events via webhooks
- **Figma Trigger (Beta)** (`figmaTrigger`) - Starts the workflow when Figma events occur
- **Form.io Trigger** (`formIoTrigger`) - Handle form.io events via webhooks
- **Formstack Trigger** (`formstackTrigger`) - Starts the workflow on a Formstack form submission.
- **Gumroad Trigger** (`gumroadTrigger`) - Handle Gumroad events via webhooks
- **Jotform Trigger** (`jotFormTrigger`) - Handle Jotform events via webhooks
- **Local File Trigger** (`localFileTrigger`) - Triggers a workflow on file system changes
- **Manual Trigger** (`manualTrigger`) - Runs the flow on clicking a button in n8n
- **n8n Trigger** (`n8nTrigger`) - Handle events and perform actions on your n8n instance
- **Postmark Trigger** (`postmarkTrigger`) - Starts the workflow when Postmark events occur
- **SSE Trigger** (`sseTrigger`) - Triggers the workflow when Server-Sent Events occur
- **SurveyMonkey Trigger** (`surveyMonkeyTrigger`) - Starts the workflow when Survey Monkey events occur
- **Toggl Trigger** (`togglTrigger`) - Starts the workflow when Toggl events occur
- **Typeform Trigger** (`typeformTrigger`) - Starts the workflow on a Typeform form submission
- **Webhook** (`webhook`) - Starts the workflow when a webhook is called
- **Workable Trigger** (`workableTrigger`) - Starts the workflow when Workable events occur
- **Workflow Trigger** (`workflowTrigger`) - Triggers based on various lifecycle events, like when a workflow is activated
- **Wufoo Trigger** (`wufooTrigger`) - Handle Wufoo events via webhooks

## Usage

```bash
cd examples/nodes/trigger
terraform init
terraform plan
terraform apply
```
