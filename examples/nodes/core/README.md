# Core Nodes Examples

This directory contains examples for **Core** category nodes.

Total nodes in this category: **5**

## Nodes in this Category

- **Code** (`code`) - Run custom JavaScript or Python code
- **If** (`if`) - Route items to different branches (true/false)
- **Merge** (`merge`) - Merges data of multiple streams once data from both is available
- **Set** (`set`) - Add or edit fields on an input item and optionally remove other fields
- **Switch** (`switch`) - Route items depending on defined expression or rules

## Usage

```bash
cd examples/nodes/core
terraform init
terraform plan
terraform apply
```
